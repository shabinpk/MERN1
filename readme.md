# Goalsetter MERN app

This is a MERN stack application that allows users to create goals, set milestones for each goal

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
